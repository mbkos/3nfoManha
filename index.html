Versionamento

O Git é um software que permite realizar versões de um projeto, onde o Github Desktop é um software que faz a ligação entre o VS Code, Git e Github.

O Github desktop permite fazer a sincronia entre o repositório local e o repositório remoto

Commit

É o precesso de confirmação de uma versão entre vírgulas, ou seja, a cada alteração é possivel confirmar a alteração

Push

Esse comando envia (sincronia) a alterações realizadas no repositorio local com o repositorio remoto com o GitHub
